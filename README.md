# Product Management CRUD

A responsive CRUD web application built with HTML, CSS, and JavaScript. The project allows users to manage products by creating, reading, updating, deleting, and searching records with data stored in Local Storage.

## Features

- Create Products
- Read Products
- Update Products
- Delete Products
- Search Products
- Local Storage Persistence
- Responsive User Interface

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## Purpose

This project was developed to practice JavaScript fundamentals, DOM manipulation, CRUD operations, and client-side data storage using Local Storage.
